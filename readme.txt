# Janith's Asset Management System

Sir,
This is a Web-Based Asset Management System built with Python (Flask) and SQLite.

## Features
- Add assets
- Edit assets
- Delete assets
- Search assets
- Print asset list

## How to Run

1. Install Python
2. Extract this project folder
3. Create a virtual environment - python -m venv venv
4. Activate it: Windows: `.\venv\Scripts\activate`
		Mac/Linux: `source venv/bin/activate`
5. Install dependencies: pip install -r requirements.txt
6. Run the app: python app.py
7. Open browser and go to: http://127.0.0.1:5000/

## Notes
- `assets.db` is the database file. You can start with sample data or empty.
